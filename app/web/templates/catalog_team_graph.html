Step 2. Feature flags

Merge node chain threshold
How merge chain threshold works
Selects the minimum size of consecutive shared procedures to count as one merge chain.
  • 0: merge node detection is fully disabled.
  • 1: each shared procedure is treated as a merge node.
  • N > 1: only non-overlapping strictly linear chains of at least N shared procedures are counted.
  • Cycles are excluded from merge-chain detection.
  • Branch/fork and join procedures are treated as chain boundaries for N > 1.
